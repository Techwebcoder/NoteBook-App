# 📒 iNotebook - MERN Notes App

iNotebook is a full-stack notes management web application built using the **MERN Stack** (MongoDB, Express.js, React.js, Node.js).

It allows users to securely create an account, log in, and manage personal notes with full CRUD operations.

---

# 🚀 Features

✅ User Registration  
✅ User Login Authentication  
✅ JWT Token Security  
✅ Password Hashing using bcryptjs  
✅ Create Notes  
✅ Read Notes  
✅ Update Notes  
✅ Delete Notes  
✅ Protected Routes  
✅ Responsive UI  
✅ MongoDB Database Storage
